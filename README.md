# Blog Challenge

### This project worked with the following stack:

- React
- Typescript
- Vite
- Mui
- Styled components
- Jest
- testing library

##### Installation

- Run in the repository root
  ```bash
  npm i
  ```

##### Run Tests

- Run for all project
  ```bash
  npm run test
  ```
- Run for a specific file
  ```bash
  npm run test -- <Component Name>
  ```

##### Run Proyect

- Run for client
  ```bash
  npm run dev
  ```
- Run for api
  ```bash
  npm run api
  ```
